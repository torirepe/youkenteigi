2. 基本要件

2.1 概要

背景

クライアントである源氏山クリニックは現在まで WEB サイトを持たず、昔ながらの口伝えなどで顧客を獲得していた。 現在ご子息が分院を予定しており、関連医院として両医院のブランディングを図るため、(株)red.sadist に WEB サイトの新規作成をご発注。
(株)red.sadist から、(有)KeoChii ほか数社にデザイン作成の打診をいただきました。

業務概要

源氏山クリニックWEBサイトの新規作成に際する、企画・構成およびデザイン。
源氏山クリニックは鎌倉市源氏山公園付近に医院をかまえ、周辺住民を顧客とする昔ながらの歯科医院である。今回、ご子息の分院を控えて親医院となるクライアントをブランドとしてイメージアップする狙いで WEB サイトを新規作 成する。

務の具体的内容

  • 企画・構成
  • Top ページおよび下層ページ(案内/アクセス)デザイン

2.2 目的

  • プロによるデザインによって、源氏山クリニックのイメージアップを図り、周辺医院に対するブランディングを確立 すること
  • WEB サイトから情報を発信することによって、鎌倉市周辺に流入してくる新規顧客を獲得すること

2.3 ターゲット

ターゲットユーザとしては、主に以下の 2 つを設定する

  • 鎌倉市に流入してくる、20~40 代女性
  • 上記の子供であり、WEB サイトを閲覧すると思われる 10~15 歳の男女

そのほか、鎌倉市住民はすべてユーザとなる可能性があるとする

2.4 コンセプト

  • 女性を意識した、やわらかく親しみの持てる雰囲気
  • 病院にふさわしく、明るく清潔感のあるイメージ
  • 高齢の住民や、コアターゲットの子供にあたる10代前半の子供が閲覧する可能性を考慮し、読みやすく分かりやすい画面構成

2.5 制約事項

代理店(株)red.sadist による他社との比較・精査の期間があるため、納品期限は厳守。

3. 機能要件

3.1 動作環境等に関する条件

対応 OS/ブラウザ

  • Windows・Mac
    • Netscape 4.7 以上
    • InternetExplorer 4.0 以上
  • CATV

対応モニタ

  • 800*600 以上
  • 16,000 色以上

ページレイアウトサイズ

  • 対応モニタサイズに準ずる(最大幅 700 ピクセル程度)

サイトを構成する大項目リスト

大項目 概要 必要要素
TOP ページ サイトの顔として、キャッチやトピックスなどを用いた 医院紹介や、全項目へのリンクを設ける。 ・ 全コンテンツへのリンク
・ ホワイトニングなどのトピックス
・ 院長からのメッセージ
案内/アクセス 来訪ユーザが医院を訪れるために必要な情報を提 供する ・ 地図および住所、電話番号
・ 医院までの道順
・ 診療時間表
診療案内 医院の診療内容に関する情報を提供する ・ 診療項目案内
・ 診療時間表
・ 医師の紹介
歯列矯正について 歯列矯正についての紹介を行う ・ 歯列矯正のメリットや治療方法な どの紹介
ホワイトニングについて ホワイトニングについての紹介を行う ・ ホワイトニングのメリットや治療方 法の紹介
その他の歯科治療 その他の歯科治療についての紹介を行う ・ その他の治療内容の紹介
お問い合わせ ユーザが医院に問い合わせをするために必要な 情報を提供する ・ 医院の住所、電話番号、E-mail アドレス
・ 営業時間
